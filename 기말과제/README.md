# 영어 일기 어플리케이션
---
## 프로젝트 소개
일상을 영어로 간단하게 기록할 수 있는 어플리케이션

## 주요 기능
* 비회원: 회원가입
* 회원: 로그인, 로그아웃, 회원 탈퇴, 일기 작성, 프로필 확인
